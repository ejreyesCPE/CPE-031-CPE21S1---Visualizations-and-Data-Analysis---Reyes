## 📝 Course Description
This repository serves as a centralized portfolio for all laboratory activities, projects, and assignments completed for **CPE 031 (Visualizations and Data Analysis)**. 

The focus of this course revolves around:
* **Data Processing:** Cleaning, structuring, and preparing datasets for analysis.
* **Exploratory Data Analysis (EDA):** Discovering patterns, spotting anomalies, and checking assumptions using summary statistics.
* **Data Visualization:** Crafting meaningful graphical representations to communicate data-driven insights effectively.
